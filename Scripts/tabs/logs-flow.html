Flux Logs v1.03

Utilisation
  • Double‑cliquez sur le canevas pour ajouter une boîte.
  • Chaque boîte doit avoir un Label et un Point de passage au format AAAA[999] (3‑4 lettres + 3‑4 chiffres entre crochets).
  • Glissez les boîtes pour les déplacer.
  • Pour relier : cliquez le point bas d’une boîte (sortie), puis le point haut d’une autre (entrée).
  • Après chargement d’un CSV, le compteur de chaque boîte affiche le nombre d’interactions passées par ce point.
