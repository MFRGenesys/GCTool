Comment utiliser :
  1. Glissez une boîte de la palette vers le canvas
  2. Cliquez sur une boîte pour la configurer
  3. Cliquez sur les points de connexion pour relier les boîtes
  4. Configurez les conditions de routage
  5. Générez les visuels pour tester le flux
Types de boîtes :
  • Start Point de départ
  • Route Passage simple
  • Menu Branchement multiple
  • Message Diffusion d'un message
  • Calendar Calendrier d'ouverture
  • End Point d'arrivée
  • Error Point d'erreur
