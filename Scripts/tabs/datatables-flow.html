Comment utiliser :
  1. Glissez une boite de la palette vers le canvas
  2. Cliquez sur une boite pour la configurer
  3. Cliquez sur les points de connexion pour relier les boites
  4. Configurez les conditions de routage
  5. Generez les visuels pour tester le flux
Types de boites :
  • Start Point de depart
  • Route Passage simple
  • Menu Branchement multiple
  • Message Diffusion d'un message
  • Calendar Calendrier d'ouverture
  • End Point d'arrivee
  • Error Point d'erreur
