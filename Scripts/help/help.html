Aide configuration OAuth Genesys

Pour GCTool, configurez une OAuth Client dans Genesys Cloud puis copiez le Client ID.

Implicit Grant

Grant Type: Token Implicit Grant (Browser)

Authorized redirect URI: https://mfrgenesys.github.io/GCTool/

Copiez le Client ID dans le champ Client ID de cette page.

PKCE

Grant Type: Code Authorization / PKCE

Authorized redirect URI: https://mfrgenesys.github.io/GCTool/

Copiez ensuite le Client ID dans GCTool.
